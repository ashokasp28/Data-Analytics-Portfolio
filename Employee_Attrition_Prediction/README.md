# Employee Attrition Prediction using Machine Learning

## 📌 Project Overview

Employee attrition is one of the biggest challenges faced by organizations. This project predicts whether an employee is likely to leave the company using machine learning techniques.

---

## 🎯 Objective

Build a classification model capable of predicting employee attrition based on employee-related features.

---

## 📂 Dataset

IBM HR Analytics Employee Attrition Dataset

Dataset includes information such as:

- Age
- Department
- Job Role
- Monthly Income
- Years at Company
- Overtime
- Education
- Job Satisfaction
- Attrition

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 📊 Project Workflow

1. Data Loading
2. Data Exploration
3. Data Preprocessing
4. Feature Scaling
5. Train-Test Split
6. Logistic Regression Model
7. Model Evaluation

---

## 🤖 Machine Learning Algorithm

- Logistic Regression

---

## 📈 Evaluation Metrics

- Accuracy
- Confusion Matrix
- Classification Report

Accuracy Achieved:

**87.76%**

---

## 📌 Project Outcome

The Logistic Regression model successfully predicts employee attrition with good accuracy and can assist HR departments in identifying employees who may be at risk of leaving.
